# Arquitetura Lógica do Sistema

O sistema segue arquitetura sequencial com decisões condicionais.

## Etapas

1. Coleta de fontes
2. Normalização
3. Validação
4. Filtragem
5. Classificação
6. Priorização
7. Exportação

## Lógica de decisão

- Score ≥70 → Rigoroso
- Score 60–69 → Oportunidade
- Score <60 → Excluído
