# MANUAL DE INSTALAÇÃO — SISTEMA INTELIGENTE DE VAGAS

Versão: 4.0

---

## VISÃO GERAL
Este sistema é um agente de IA projetado para buscar, filtrar e priorizar vagas de emprego automaticamente com base em critérios personalizados.

---

## REQUISITOS
Para utilizar o sistema é necessário:

- acesso a um chat com IA
- arquivo Core Prompt
- arquivo Config

---

## PASSO A PASSO DE INSTALAÇÃO

### 1. Abrir o chat de IA
Abra a interface onde o agente será utilizado.

---

### 2. Inserir o Core Prompt
Copie todo o conteúdo do arquivo:

core_prompt_siv.md

e envie como primeira mensagem.

---

### 3. Inserir o arquivo de configuração
Copie o conteúdo de:

config_siv.json

e envie na conversa.

---

### 4. Ativação
Digite:

ATIVAR SISTEMA

---

## COMO USAR
Depois de ativado, basta solicitar buscas de vagas normalmente.

Exemplos:

- buscar vagas
- atualizar vagas
- vagas recentes

---

## FUNCIONAMENTO INTERNO
O sistema executa automaticamente:

- filtragem
- validação
- classificação
- priorização
- organização

---

## PERSONALIZAÇÃO
Para alterar critérios:

- editar config_siv.json
- reenviar no chat

---

## SOLUÇÃO DE PROBLEMAS

Se o sistema não responder corretamente:

1. reenviar config
2. reenviar core
3. ativar novamente

---

## RECOMENDAÇÃO
Sempre utilizar a versão mais recente dos arquivos.

---

FIM DO MANUAL
