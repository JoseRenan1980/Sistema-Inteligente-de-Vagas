# Padrão Oficial de Tabelas — SIV v3.0

## Estrutura obrigatória
Todas as tabelas devem conter exatamente as colunas abaixo:

1. Classificação
2. Score de Prioridade
3. Função
4. Empresa
5. Setor / Porte
6. Tipo de Contrato
7. Modalidade
8. Bairro
9. Distância (km)
10. Tempo de Deslocamento
11. Salário
12. Benefícios
13. ATS Match %
14. Palavras-chave
15. Perfil Comportamental
16. Nível de Concorrência
17. Data da Divulgação
18. Vagas 24h
19. Vagas 72h
20. Vagas 7 dias
21. ID da Vaga
22. Link de Acesso
